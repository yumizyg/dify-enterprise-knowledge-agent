## 导入 Dify 知识库建议

1. 在 Dify **知识库** 中创建两个数据集：
   - **流程库**：上传 `制度性文件【虚拟数据】/` 下全部 docx
   - **产品库**：上传 `产品数据【虚拟数据】/` 下全部 xlsx
2. 导入 DSL 工作流后，在对应「知识检索」节点重新绑定上述两个数据集
3. 切片建议：
   - docx：按段落 / 标题层级切分，保留步骤编号
   - xlsx：按行或按 Sheet 切分，确保 ASIN、SKU 等编码字段完整保留
